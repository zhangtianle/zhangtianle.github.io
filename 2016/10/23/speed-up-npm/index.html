加速npm

发表于 2016-10-23 | 分类于 应用开发 |

前一段时间由于hexo出了莫名其妙的问题一直无法解决，所以打算重新安装。但是由于GFW的原因，npm的下载速度简直无法忍受，之前买的VPN也不知道什么原因被屏蔽了。在忍无可忍之时，在网上百度到了npm的国内淘宝镜像。

taonpm

淘宝npm项目地址TAONPM

这是一个完整 npmjs.org 镜像，你可以用此代替官方版本(只读)，同步频率目前为 10分钟 一次以保证尽量与官方服务同步。

使用说明

使用定制的 cnpm (gzip 压缩支持) 命令行工具代替默认的 npm:

安装完毕之后，我们就可以直接使用cnpm来替代原来的npm来进行安装啦~(≧▽≦)/~

从 registry.npm.taobao.org 安装所有模块. 当安装的时候发现安装的模块还没有同步过来, 淘宝 NPM 会自动在后台进行同步, 并且会让你从官方 NPM registry.npmjs.org 进行安装. 下次你再安装这个模块的时候, 就会直接从 淘宝 NPM 安装了.

OK 搞定！
