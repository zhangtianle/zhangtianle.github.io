关于我

Talk is cheap. Show me the code.

小学开始接触电脑。从电话线嘟嘟嘟的拨号方式连接互联网，那时候的网费贵的一批，而且不能同时打电话与上网。当时印象最深的当属三大门户网站（新浪、搜狐、网易）。

初一开始接触网络游戏。第一个当属CS，接下来是传奇世界、梦幻西游、奇迹等等。盛大在我心中是具有传奇般色彩的公司。因为在家偷偷打游戏，被爸妈打骂 n 次（已经数不清）。期间发挥各种想象，采取了各种对抗策略。心中还有个黑客梦想，在网上搞了木马，结果自己把自己的QQ号给盗了。

高一开始接触编程。由于中学生竞赛国家奖高考有加分并且自己感兴趣，因此同时报了两，物理和信息竞赛，周内晚上、周末白天学校统一集训。写的还是 Free Pascal，而 C 和 Java 对我来说简直是个神仙，最终迫于学业压力以及竞赛拿奖无望，后放弃。但此经历奠定了我初始的编程基础，同时了解了计算机神奇的算法。

大一开始接触开发。由我的 C 语言老师领我进入工程的大门，在实验室师兄的带领下学习 Java 开发。主攻后端开发，了解前端知识。随后开始接项目，与客户谈需求，带实验室同学开发小型项目。

研一开始接触算法。被研究生导师 diss 工程后，转入算法领域（最早可以追溯到本科的数模比赛）。研究社交网络，影响力最大化问题，后产出一篇论文、一篇专利，但被拒 n 次，毕业时仍未发表。

毕业后，第一份工作进入阿里巴巴，专注淘系营销策略业务。未来已来~
